# Software Bug Prediction using Machine Learning

## Project Overview

This project predicts whether a software module is likely to contain defects using software quality metrics.

## Dataset

Eclipse Bug Prediction Dataset

Dataset Link:
https://www.kaggle.com/datasets/syedzubair/bug-prediction-dataset

## Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Handling
4. Feature Engineering
5. Feature Scaling
6. Train-Test Split
7. Model Training
8. Hyperparameter Tuning using Optuna
9. Model Evaluation

## Models Used

- Logistic Regression
- Extra Trees Classifier
- CatBoost Classifier

## Best Result

- Accuracy: 86%
- Best Model: CatBoost Classifier
